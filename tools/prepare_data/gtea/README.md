# Data Preparation for GTEA

## Download Annotations
Please download the annotation `data/annotations/` from [Box](https://oregonstate.box.com/s/fyxgm3a4vk5qtzygn2zot85ce48y2b9l).

The annotation should be in `data/annotations/`.

## Download Raw Videos

Please put the downloaded video under the path: `data/gtea/raw_data/video/`.

You can download the raw video from [official website](https://cbs.ic.gatech.edu/fpv/), or download from this [Box](https://oregonstate.box.com/s/fyxgm3a4vk5qtzygn2zot85ce48y2b9l).
