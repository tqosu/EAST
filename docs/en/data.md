# Prepare the Annotation and Data

The following table lists the supported datasets and provides links to the corresponding data preparation instructions.

| Dataset                                                    | Description                                                                                   |
| :--------------------------------------------------------- | :-------------------------------------------------------------------------------------------- |
| [GTEA](../../tools/prepare_data/gtea/README.md)   | GTEA consists of 28 egocentric videos, annotatedwith 11 action classes. |
| [50Salads](/tools/prepare_data/50salads/README.md)   | 50Salads is a collection of 50 top-view videos of salad preparation with 17 action classes. |
| [Breakfast](/tools/prepare_data/breakfast/README.md)   | Breakfast consists of 1712 videos showing 48 breakfast preparation actions from a third-person perspective. |
| [Assembly101](/tools/prepare_data/assembly101/README.md)   | Assembly101 consists of 4321 videos and 202 action classes based on 11 verbs and 61 objects. |
